Performing your daily tasks in VirtualHome

We want to study how people typically do tasks at home. For that, you will control a character in a virtual environment, and your task is to drive the character to finish the tasks we ask you. You will need to explore the environment to find the objects needed for the task, grab them and place them around the environment to complete the task we ask you

The below link will walk you through a few tasks in the Virtual Home Environment. You will start with a quick tutorial. Once you are familiar with the environment or have completed the tutorial, skip forward to the next task

Below is an example of the environment and the commands to interact with the environment, familiarize yourself before you start. (They will be repeated on the next webpage)

Commands

Click on objects in the screen to interact with them. Buttnos will appear when you can interact with those objects. If a button does not appear, you may need to get closer to the object to interact with it.
  • W, A, S, D: Navigate arround the environment (forward, badkward, rotate left, rotate right)
  • Up arrow key: Look up
  • Down arrow key: Look down
  • V: Save Current Episode
  • G: Finish this task
  • G: Finish this episode
  • Left-click on screen: Select object to interact
  • Q (with object selected): rotate object counterclockwise
  • E (with object selected): rotate object clockwise
